************************************************************************************************
*                                                                                              *
*                                        INSTALLATION:                                         *
*                                                                                              *
* 1.) Extract to a folder of your choice                                                       *
* 2.) Put the hashed classfile(s) and this tool into same folder                               *
* 3.) Run either SASClassResolver.exe of SASClassResolver.jar, see details below               *
* 4.) Have fun !                                                                               *
*                                                                                              *
************************************************************************************************
*                                                                                              *
*                                          CREDITS:                                            *
*                                                                                              *
* - SAS Team: Da whole thing!                                                                  *
*                                                                                              *
************************************************************************************************
*                                                                                              *
* Usage:                                                                                       *
*                                                                                              *
* You can run this tool either the "classic" way:                                              *
*   java -jar SASClassResolver.jar [<path of hashed files> or <hashed file>]                   *
* Or you can run the precompiled executable directy:                                           *
*   SASClassResolver.exe [<path of hashed files> or <hashed file>]                             *
*                                                                                              *
* If path/file is omitted, current path will be used.                                          *
*                                                                                              *
* For quick&dirty operation, just drop SASClassResolver.exe into the folder where your hashed  *
* classfiles are, and run it by double-clicking on that executabe file.                        *
*                                                                                              *
************************************************************************************************
*                                                                                              *
* Examples:                                                                                    *
*                                                                                              *
* The following commands resolve all hashed classfiles in the current folder,                  *
*  the result will be exaclty the same for each of these commands:                             *
*   SASClassResolver.exe                                                                       *
*   SASClassResolver.exe .                                                                     *
*   java -jar SASClassResolver.jar                                                             *
*   java -jar SASClassResolver.jar .                                                           *
*                                                                                              *
* The following commands resolve the hashed classfile "01234567890ABCDEF" only,                *
*  the result will be exaclty the same for both of these commands:                             *
*   SASClassResolver.exe 01234567890ABCDEF                                                     *
*   java -jar SASClassResolver.jar 01234567890ABCDEF                                           *
*                                                                                              *
* The following commands resolve all hashed classfile from the "hashed" subfolder,             *
*  the result will be exaclty the same for both of these commands:                             *
*   SASClassResolver.exe hashed                                                                *
*   java -jar SASClassResolver.jar hashed                                                      *
*                                                                                              *
